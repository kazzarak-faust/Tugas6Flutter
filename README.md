# tugas6

Pebjelasan:  
1. User mengisi data pada halaman FormDataPage, lalu data ditampung menggunakan TextEditingController,
2. Ketika tombol "Tampilkan Data" ditekan, maka Navigator.push() digunakan untuk berpindah ke halaman lain,  
3. Pada Navigator.push, kita memanggil halaman TampilDataPage sambil mengirimkan data yang user isi pada halaman FormDataPage,
4. Di TampilDataPage, data yang dikirim disimpan sebagai variabel final di constructor final String,
5. Variabel tersebut kemudian ditampilkan dalam Widget.
